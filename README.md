Vue 3 + Vite + Antigravity

Project ini dibuat menggunakan kombinasi modern stack: Vue 3, Vite, dan Antigravity untuk membantu proses development menjadi lebih cepat, terstruktur, dan efisien. Dengan dukungan Vue 3 Composition API serta build tool Vite yang ringan dan super cepat, project ini dirancang agar mudah dikembangkan maupun di-scale untuk kebutuhan aplikasi modern.

Antigravity digunakan sebagai fondasi workflow dan struktur pengembangan sehingga proses pembuatan UI, manajemen komponen, dan pengelolaan project menjadi lebih rapi serta produktif.
